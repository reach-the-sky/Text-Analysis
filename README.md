### Installation
> Download __[Glove 6B](https://nlp.stanford.edu/data/glove.6B.zip)__ data and place it in this file structure.
```
|--data
   |--glove.6B
      |--glove.6B.100d.txt
```

```
$ pip install requirements.txt
```
